## 概要
- streamlitのページ遷移について検証。

## 検証結果
### サイドバーによるページ遷移
- ページの遷移をサイドバーから行える。
- ページのディレクトリパス、名前、アイコン、URLのパスを設定ファイルに逃がすことができる。（アイコンは設定しなければ消すこと可能）
- ~~ただし、URLにパスを入力しても遷移しない。~~
